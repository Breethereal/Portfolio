## Projects

### 1. Credit Risk Modelling: Logistic Regression vs XGBoost

Developed an end-to-end credit risk model using the German Credit dataset to classify loan applicants as high-risk or low-risk borrowers.

- Built full machine learning pipeline in R  
- Applied one-hot encoding for structured financial data  
- Compared Logistic Regression and XGBoost models  
- Evaluated performance using AUC (~0.77), sensitivity, and specificity  
- Performed threshold tuning (0.3–0.5) to optimize decision outcomes  

**Key Insight:**  
Threshold selection had a greater impact on business outcomes than model complexity.

**Tools:** R, caret, xgboost, pROC, ggplot2  
**View Project:** https://github.com/Breethereal/credit-risk-modelling-logreg-vs-xgboost

### 2. Fraud Detection Using Machine Learning

Built a classification model to identify fraudulent transactions while balancing detection rate and false positives.

- Preprocessed transaction data and handled class imbalance  
- Trained classification models for fraud detection  
- Focused on maximizing recall (capturing fraudulent activity)  
- Evaluated trade-offs between detection accuracy and operational efficiency  

**Key Insight:**  
High recall is critical in fraud detection, but must be balanced to avoid excessive false alerts.

**Tools:** Python, Machine Learning, Data Processing
